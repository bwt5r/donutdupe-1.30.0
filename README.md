📦 DonutSMP Item Duplicator Mod

A lightweight and experimental Minecraft mod designed to explore item duplication mechanics on the DonutSMP server.

This project demonstrates how specific in-game behaviors and edge cases can be leveraged to duplicate items efficiently. Built for educational and research purposes, the mod focuses on simplicity, performance, and ease of use.

✨ Features

- ⚡ Fast and reliable item duplication methods

- 🧩 Simple integration with existing gameplay

- 🔧 Easy-to-configure settings

- 🚀 Optimized for minimal performance impact

⚠️ Disclaimer

This project is intended for educational purposes only. Usage on servers without permission may violate their rules and could result in penalties.
